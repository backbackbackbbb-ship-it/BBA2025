About BBA2025

BBA2025 is a classroom project built to help first-year BBA students understand how a real online shopping platform is designed and built — from the homepage banner down to the checkout button.

Every product, price and order on this site is fictional and exists purely for learning HTML, CSS and JavaScript concepts such as DOM manipulation, event handling and responsive design.

No real payments are processed and no real orders are shipped — this is a 100% educational simulation.
